# 🐾 Topvet – Veterinar Clinic Website

Official website for **Topvet**, a professional veterinary clinic. The site is built as a modern, fast, and responsive single-page application designed to present the clinic's services, showcase the team, and allow potential clients to get in touch.

---

## ✨ Features

- **Hero Section** – A compelling introduction to the clinic with a clear call-to-action.
- **About Us** – The story and values behind Topvet.
- **Services** – A detailed overview of the veterinary services offered.
- **Gallery** – Photo gallery showcasing the clinic and its environment.
- **Testimonials** – Real reviews from satisfied clients.
- **Contact / CTA** – An integrated contact form connected to Supabase for lead capture.
- **Footer** – Business info, social links, and quick navigation.

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/) | UI framework |
| [Vite](https://vitejs.dev/) | Build tool & dev server |
| [Supabase](https://supabase.com/) | Backend & form submissions |
| CSS Modules / Custom CSS | Styling |

---

## 🚀 Getting Started

**Install dependencies:**
```bash
npm install
```

**Start the development server:**
```bash
npm run dev
```

**Build for production:**
```bash
npm run build
```
